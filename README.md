# bob-webinterface

## Setup global dependencies (optional, to use global `vue` in bash)

```
sudo yarn global add @vue/cli
```

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```

### Lints and fixes files
```
yarn run lint
```
